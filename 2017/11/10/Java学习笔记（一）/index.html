Java笔记

一、Java 简介

1.1995sun公司正式发布java第一个版本
2.Java开发工具：

a.jcreator

b.jbuilder

c.netbean

d.eclipse

3.JDK：

a.jre（java运行环境）

b.java工具包（Java编译器javac.exe 解释执行器java.exe

c.Java类库（常用150多个）

4.jre是java运行环境，运行java需要配置jre环境
5.jdk配置：添加环境变量（见附一win10环境配制）

二、java方法

1.方法：完成特定功能的代码块
2.方法就是函数
3.方法格式：
修饰符 返回值类型 方法名（参数类型 参数名1，参数类型 参数名2…）{
方法体语句
Return 返回值
}
4.详细解释：

a.修饰符

b.返回值类型：功能结果的数据类型

c.方法名：符合命名规则即可，方便我们调用

d.参数：

实际参数：实际参与运算

形式参数：方法定义上的，用于接收实际参数

e.参数类型：参数的数据类型

f.参数名：就是变量名

g.方法体语句：就是完成功能的代码

h.Return：结束方法

i.返回值：功能的结果，有return带给调用者

5.方法的执行特点：不调用，不执行
6.方法的调用方式：

a.单独调用：

b.输出调用

c.复制调用

举例：
    //需求：求两个数据之和的案例
class FunctionDemo {
    public static void main(String[] args) {
        int x = 10;
        int y = 20;

        //方式1：单独调用
        //sum(x,y);

        //方式2：输出调用
        //System.out.println(sum(x,y));

        //方式3：赋值调用
        int result = sum(x,y);
        //result在这里可以进行操作
        System.out.println(result);
    }
public static int sum(int a,int b) {
        return a + b;
    }
}
7.方法的注意事项：

a.方法与方法是平级关系，不能嵌套使用

b.如果方法有明确的返回值，需用return带回一个值

8.方法重载的特点：

a.与返回值无关，只看方法名和参数列表

b.在调用时，虚拟机通过参数列表的不同来区分同方法

三、面向对象

1.面向对象：面向对象是基于面向过程的编程思想
2.面向对象的思想特点：

a.是一种更符合我们思考习惯的思想

b.把复杂的事情简单化

c.让我们从执行者变成了指挥者

3.类与对象:

a.现实世界的事物
属性 事物的基本描述
行为 事物的功能
b.Java语言中最基本的单位是类。所以，我们要用类来体现事物

c.类
成员变量 事物属性
成员方法 事物行为

d.类：是一组相关的属性和行为的集合。是一个抽象的概念。
对象：是该类事物的具体存在，是一个具体的实例。(对象)

举例：
  学生：类
  班长：对象
4.类的定义及使用

a.类的定义
成员变量 定义格式和以前一样，就是位置不同，在类中，方法外。
成员方法 定义格式和以前一样，就是去掉了static。

b.使用类的内容

·创建对象? 格式
类名 对象名 =  new 类名();

·使用成员变量和成员方法
    对象名.成员变量
    对象名.成员方法()
